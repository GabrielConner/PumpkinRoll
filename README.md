# Pumpkin Pie

3D project framework for my projects.
Handles the placing of objects as well in an admin scene.





## To use this project you must include :



### External Packages :

* glad
* glfw
* glm







### Personal Packages :

* timer
* vector
* windowManager
* fileInterface


Prone to change as I adjust the packages to fit better for when I turn the project into a DLL
